# Links

Attach external URLs to nodes for quick reference.

## Overview

The Links section in the InfoPanel lets you save external URLs with titles and descriptions.

## Adding Links

1. Click a node to open InfoPanel
2. Scroll to **Links** section
3. Click **+ Add Link**
4. Enter:
   - **URL** (required)
   - **Title** (optional)
   - **Description** (optional)
5. Click **Save**

## Managing Links

- **Open**: Click any link to open in new tab
- **Edit**: Hover and click edit icon
- **Delete**: Hover and click trash icon

## Use Cases

- Reference documentation
- Related articles
- Source material
- External tools
- Project URLs

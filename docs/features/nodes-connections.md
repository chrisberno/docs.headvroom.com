# Nodes & Connections

The core building blocks of your knowledge graph.

## Nodes

Nodes are the fundamental units of your graph. Each node represents an idea, topic, resource, or concept.

### Creating Nodes

1. Click **+ Add Node** in the toolbar
2. Enter a title (required)
3. Choose an icon: Home, Folder, Bookmark, or Idea
4. Select a color (optional)
5. Click **Create Node**

### Editing Nodes

- **Title**: Double-click to edit inline
- **Content**: Click node to open InfoPanel, then edit notes, links, media
- **Icon/Color**: Currently set at creation (editing coming soon)

### Deleting Nodes

1. Click the node to select it
2. Click **Delete** or press the Delete key
3. Confirm deletion

:::warning
Deleting a node also removes all its connected edges.
:::

---

## Edges (Connections)

Edges are the lines connecting nodes. They represent relationships between concepts.

### Creating Edges

1. Click **Connect** button in toolbar (enables connection mode)
2. Hover over source node to see handles (circles on edges)
3. Click and drag from a handle
4. Drop on target node's handle
5. Edge is created and saved

### Deleting Edges

1. Click an edge to select it (highlights)
2. Click **Delete** or press Delete key
3. Confirm deletion

## Graph Navigation

### Focus Mode

Click any node to **focus** it:
- Node centers on screen
- Connected nodes arrange around it
- InfoPanel opens on right

### Exploration

- **Expand**: Click unfocused nodes to explore their neighborhoods
- **Breadcrumbs**: Track your path at the top of the screen
- **History**: Click any breadcrumb to jump back

### Canvas Controls

| Action | How |
|--------|-----|
| Pan | Drag on empty canvas |
| Zoom | Scroll wheel or pinch |
| Fit view | Double-click empty canvas |
| Minimap | Bottom-right corner shows viewport |

## Graph Operations Summary

| Operation | How |
|-----------|-----|
| **Create node** | + Add Node button |
| **Edit title** | Double-click node |
| **Delete node** | Select → Delete button or key |
| **Create edge** | Connect button → drag handle to handle |
| **Delete edge** | Click edge → Delete |
| **Focus node** | Single click |
| **Navigate back** | Breadcrumb trail |

Users Guide

Everything you need to know to use Headvroom effectively.

What is Headvroom?​

Headvroom is a visual knowledge management tool. Instead of folders and files, you organize information as a graph of connected nodes.

Core Concepts​

Nodes​

Individual items in your graph - ideas, topics, resources, projects, or anything else you want to track.

Edges​

Connections between nodes. They show relationships and let you navigate by clicking.

InfoPanel​

The sidebar that opens when you click a node. Contains notes, links, files, and more.

Quick Reference​

TaskHow
Create nodeClick + Add Node
Edit node titleDouble-click the title
Delete nodeSelect → Delete button
Connect nodesClick Connect → drag between handles
Add notesClick node → type in Notes section
Add linksClick node → + Add Link
Upload filesClick node → Media → drag files
Change themeClick palette icon in header
Change layoutClick layout dropdown in toolbar

  • All data stored in your Supabase database
  • Auto-saves as you work
  • Syncs across devices (same account)
  • You control sharing via embed settings
